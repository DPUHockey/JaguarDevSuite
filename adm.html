User Guide
How To Use Jaguar ADM
A complete game day roster management tool for the Fronek Skillzone Hockey program. Build rosters, split teams, designate goalies, and print professional lineups in minutes.
Pages Overview
🏠
Home
Landing page showing total player counts. Use the Import Roster via CSV card to bulk-upload players from a spreadsheet file.
📋
Manage
Add players manually, set the game date, drag players into Black or Gold teams, edit names, and switch levels between Mini Mite, Minor Mite, and Major Mite.
📄
Rosters
Public-facing view of the complete game day lineups. Shows every active level (Mini Mite, Minor Mite, Major Mite) with Black and Gold team splits. Updates live as you manage.
🖨
Print
Clean black-and-white version optimized for printing. Click Print / Save as PDF to open the system print dialog. Use Save as PDF to share digitally.
Adding Players
Manually
  1. 1
    Go to the Manage page and enter First Name and Last Name
  2. 2
    Select the player's level — Mini Mite, Minor Mite, or Major Mite
  3. 3
    Press Enter or click + Add. The cursor returns to First Name automatically for fast entry
  4. 4
    Player appears in the Unassigned pool for their team
Via CSV Import
  1. 1
    Go to the Home page and click the CSV import card
  2. 2
    Your CSV must have a header row with columns: FirstName, LastName, Level
  3. 3
    Level values: Mini Mite/al, Minor Mite/co, or Major Mite/ea
  4. 4
    Players are added to their respective Unassigned pools automatically
CSV column order:
First,Last,Birthdate,Level,P1 Name,P1 Email,P1 Phone,P2 Name,P2 Email,P2 Phone
Example:
John,Smith,2016-04-12,Mini Mite,Jane Smith,jane@email.com,555-1234,,,
Import this on the Master List tab -- it auto-populates the Manage page's unassigned pool.
Splitting Into Black & Gold Teams
  1. 1
    On the Manage page, players start in the Unassigned pool
  2. 2
    Click and drag any player row down into the Black Team or Gold Team drop zone. The player is removed from Unassigned automatically
  3. 3
    To move between teams, drag a player from Black Team to Gold Team or vice versa
  4. 4
    To return a player to Unassigned, drag them back up to the Unassigned drop zone
Editing Players
✏️
Edit Name or Level
Click a player's name on any row to open the edit modal. Change first name, last name, or switch between Mini Mite, Minor Mite, and Major Mite. Hit Save Changes or press Enter.
🥅
Mark as Goalie
In the edit modal, check the Goalie checkbox. The player is pinned to the top of their team roster and shown with a GK badge. Uncheck to remove goalie status.
🗑
Remove a Player
Click the ✕ button on any player row to permanently remove them from all pools. Use ✖ Clear All Data at the top of Manage to wipe all rosters at once.
📅
Game Date
Use the Game Date picker on the Manage page to set the date. It appears on both the Roster view and the Print page automatically.
Roster Order & Display
Automatic sorting — rosters are always alphabetized by last name, then first name.

Goalies first — any player marked as Goalie is pinned to the top of their team list above all other players.

Display name — goalies show as LastName — Goalie on the Roster and Print pages.

Black & Gold alignment — on the Roster and Print pages, Gold Team rosters for every active level always start at the same vertical position for easy side-by-side comparison. Levels with no one assigned that week are hidden automatically.
Printing & Sharing
  1. 1
    Go to the Print tab — it shows a clean black-and-white preview of all rosters
  2. 2
    Click 🖨 Print / Save as PDF to open the system print dialog
  3. 3
    To save as a PDF: in the print dialog choose Save as PDF (or Microsoft Print to PDF on Windows) as the destination
  4. 4
    Set margins to Minimum or None in the print dialog for the cleanest output
Hosting on GitHub Pages
  1. 1
    Create a free account at github.com and make a new public repository named jaguar-adm-rosters
  2. 2
    Upload the index.html file to the repository
  3. 3
    Go to Settings → Pages, set source to main branch / root, and save
  4. 4
    Your app is live at https://YOUR-USERNAME.github.io/jaguar-adm-rosters within 60 seconds
  5. 5
    On iPhone: open in Safari → Share → Add to Home Screen. On Android: Chrome → Add to Home Screen
Note: Rosters, team assignments, and the game date are saved automatically on this device and will be here next time you open the app. Keep the CSV file handy as a backup.
