Jaguar InHouse Hockey Program
Program Guide
Everything you need to know to run the Jaguar InHouse Game Day Roster Manager — from first setup through game day printing.
🚀
Quick Start
1
Setup — Go to the Setup tab. Enter up to four team names (e.g. Quakers, Whalers, Scouts, Golden Seals), pick a color for each, enable the ones playing, and hit Save Team Configuration.
2
Import Roster — On the Home page, drag and drop your CSV file onto the import zone. Your players will auto-sort to their teams.
3
Set Game Date — Go to the Manage tab and pick the game date using the date picker at the top.
4
Schedule Games — Go to the Schedule tab. Add your Friday practices and Sunday games. Assign locker rooms per date using the Locker Rooms panel.
5
Print — Use the Print tab to print game day rosters (organized by time slot with locker rooms), or the Print Schedule button on the Schedule tab for the weekly practice + game sheet.
6
Share — On the Setup tab, click Export & Share to download a self-contained HTML file with all your data baked in. Email it to anyone — it opens as a fully functional copy.
⚙️
Setup Tab
Team Configuration
Configure up to four teams. Each team has a name, a color (click the swatch or type a hex code), and an enabled/disabled toggle. Only enabled teams appear throughout the app. Hit Save Team Configuration after any changes.
Export & Share
Downloads a standalone .html file with all data embedded — teams, rosters, schedule, locker rooms, and game date. The recipient opens it in any browser with no setup. Always export from your working copy; the exported file is a shareable snapshot.
Clear All Roster Data
Wipes all players, games, practices, locker room assignments, and game date. Team names and colors are preserved. A confirmation prompt appears before anything is deleted.
📂
CSV Import
Required Format
Your CSV must have a header row. The importer looks for these columns in any order:

number  |  first name  |  last name  |  team  |  dob

Blank fields are fine — rows import as long as the headers exist and the team name matches. DOB accepts MM/DD/YYYY or YYYY-MM-DD.
Team Matching
The team column is matched against your configured team names — case-insensitive, with punctuation ignored. If no match is found, the team is automatically created from the CSV name. You can then assign it a color in Setup.

Tip: configure your teams in Setup first with names that exactly match your CSV for best results.
📋
Manage Tab
Adding Players
Select a team using the tabs at the top. Fill in First Name, Last Name, #, and DOB, then click + Add or press Enter. Players are sorted automatically by jersey number, with goalies always pinned to the top.
Goalie Flag
Check the Goalie checkbox when adding or editing a player to mark them as a goalie. Goalies appear in italic gold with a GK badge and are always listed first on rosters and print sheets.
Editing & Deleting
Click the ✏ pencil icon to open the edit modal — change any field including the team assignment. Click ✕ to delete a player. Changes save immediately.
Game Date
The date picker at the top of the Manage page sets the global game date used throughout the app — on the Roster page, Print page, and when looking up locker room assignments.
📅
Schedule Tab
Adding Practices
Use the Add Practice (green) form on the left. Pick a Friday date, select a time (4:50 PM or 5:50 PM), check which teams are on the ice, and click + Add Practice.
Adding Games
Use the Add Game (gold) form on the right. Pick a Sunday date, select a time slot (7:00 AM through 12:50 PM), choose the home and away teams, and click + Add Game. Format shown: Away at Home.
Locker Room Assignments
The Locker Rooms panel sits between the builders and the schedule list.

1
Pick the game date using the date picker in the panel
2
Click A B C or D next to each team to assign their locker room for that date
3
Click Apply to save — assignments are date-specific and appear as LR A badges on the game rows below
Note: Each game date can have different locker room assignments. The correct rooms show automatically on Rosters, Print, and Schedule based on the date.
Weekly View & Print Schedule
Practices and games are automatically paired side by side by week — Friday practice on the left, Sunday games on the right. Click 🖨 Print Schedule to print the current week (based on the game date set in Manage) as a clean black-and-white sheet including locker room assignments.
👥
Rosters Tab
Displays all four teams side by side in a single row. Each team column shows the team name, locker room assignment for the current game date (if set), player count, and the full roster sorted by jersey number with goalies at the top. This is a live read-only view — edit players on the Manage tab.
🖨
Print Tab
Game Day Rosters
Organizes rosters by time slot. The 7:00 AM game teams appear at the top (home and away side by side with locker rooms), and the 8:10 AM game teams appear below — separated by a dashed divider. Goalies are listed first in italics with a GK badge.
What Drives the Print
The print page is controlled by the Game Date set on the Manage tab. Make sure that date matches your scheduled games. If no games are scheduled on that date, all teams print in a standard 2-up layout as a fallback.
💡
Tips & Notes
Data is saved automatically in your browser's local storage every time you make a change — no Save button needed for roster edits.

Exporting is the only way to share data between devices or browsers. Use Export & Share on the Setup tab before emailing.

Clearing data on Setup wipes players, games, practices, locker rooms, and the game date — but keeps your team names and colors intact.

Jersey number sorting is numeric — #1 comes before #10. Blank numbers sort to the bottom.

Goalie detection works by appending a space and the letter G to the last name (e.g. Smith G). The app strips this for display and shows the clean name everywhere.

DOB is only shown on the Manage tab for reference — it does not appear on printed rosters or the Roster view.
